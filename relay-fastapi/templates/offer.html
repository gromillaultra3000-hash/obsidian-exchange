Пользовательское соглашение

Публичная оферта сервиса ObsidianExchange. Используя бот или сайт, вы принимаете эти условия.

1. Общие положения

1.1. Сервис «ObsidianExchange» (далее — Сервис) предоставляет пользователям возможность обмена рублей на криптовалюту (BTC, LTC, USDT TRC-20), обратного обмена криптовалюты на рубли и свопа криптовалют через Telegram-бота и сайт obsidian-exchange.org.

1.2. Взаимодействие с пользователем осуществляется автоматически через Telegram-бота Сервиса. По вопросам работы Сервиса пользователь может обратиться в поддержку Сервиса.

1.3. Используя Сервис, пользователь подтверждает согласие с условиями настоящего соглашения.

2. Описание услуги

2.1. Пользователь создаёт заявку в Telegram-боте или на сайте, указывая направление обмена, сумму и адрес получения.

2.2. Сервис предоставляет пользователю реквизиты для перевода рублей.

2.3. После поступления средств на указанные реквизиты Сервис отправляет пользователю указанную в заявке сумму в выбранной криптовалюте.

2.4. Курс обмена и комиссии сообщаются пользователю до совершения операции. Курс фиксируется на срок действия заявки.

2.5. При продаже криптовалюты порядок аналогичен: после поступления и подтверждения криптовалюты Сервис переводит рубли на реквизиты пользователя.

3. Ответственность сторон

3.1. Сервис не несёт ответственности за ошибки пользователя при вводе реквизитов, адресов или сумм.

3.2. Сервис не несёт ответственности за задержки, связанные с работой платёжных систем, банков или блокчейн-сетей.

3.3. Сервис не несёт ответственности за последующее использование приобретённых пользователем средств.

3.4. Перевод, выполненный с нарушением инструкции в заявке (неверная сумма, оплата по просроченной заявке, перевод с чужих реквизитов), может потребовать ручной проверки — сроки зачисления или возврата в этом случае увеличиваются.

4. Персональные данные

4.1. Сервис обрабатывает данные пользователя (Telegram ID, адреса кошельков, реквизиты для выплат) исключительно для выполнения обмена.

4.2. Данные не передаются третьим лицам.

4.3. Сервис не требует прохождения верификации личности (KYC).

5. Изменения условий

5.1. Сервис вправе изменять условия соглашения без предварительного уведомления.

5.2. Актуальная версия всегда доступна на странице obsidian-exchange.org/offer и по ссылке в боте.

6. Прочие условия

6.1. Использование Сервиса означает полное согласие пользователя с настоящим соглашением.

6.2. Все споры разрешаются путём переговоров через поддержку Сервиса.
