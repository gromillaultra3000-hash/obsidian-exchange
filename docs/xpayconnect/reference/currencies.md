# Валюты

Валюта ордера передаётся в поле `currency` при создании ордера. Должна совпадать с валютой, настроенной у мерчанта. По умолчанию используется `RUB`.

## Поддерживаемые валюты

| Код     | Символ | Валюта               | Страна / Регион |
| ------- | ------ | -------------------- | --------------- |
| **RUB** | ₽      | Российский рубль     | Россия          |
| **KGS** | сом    | Кыргызский сом       | Кыргызстан      |
| **KZT** | ₸      | Казахстанский тенге  | Казахстан       |
| **UZS** | UZS    | Узбекский сум        | Узбекистан      |
| **TJS** | с.     | Таджикский сомони    | Таджикистан     |
| **TRY** | ₺      | Турецкая лира        | Турция          |
| **INR** | ₹      | Индийская рупия      | Индия           |
| **KRW** | ₩      | Южнокорейская вона   | Южная Корея     |
| **BRL** | R$     | Бразильский реал     | Бразилия        |
| **ARS** | $      | Аргентинский песо    | Аргентина       |
| **ZAR** | R      | Южноафриканский рэнд | ЮАР             |
| **CNY** | ¥      | Китайский юань       | Китай           |

{% hint style="info" %}
Валюта фиксируется в момент создания ордера и не может быть изменена после его создания.
{% endhint %}

{% hint style="warning" %}
Передача валюты, не соответствующей настройкам мерчанта, приведёт к ошибке. Доступные для конкретного мерчанта валюты уточняйте у менеджера.
{% endhint %}
