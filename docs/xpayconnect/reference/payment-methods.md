# Платёжные методы

Используются в поле `type` при создании ордера: [PAYIN-FIAT](/orders/create.md) и [PAYIN-CRYPTO](/orders/create-crypto.md).

## Фиатные методы

| Метод           | Описание                                    |
| --------------- | ------------------------------------------- |
| **any**         | Карта РФ / СБП РФ                           |
| **card**        | Карта РФ                                    |
| **sim**         | СБП РФ                                      |
| **sngs**        | СБП трансграничный                          |
| **sngs\_abh**   | СБП Абхазия                                 |
| **sngc\_abh**   | C2C Абхазия                                 |
| **simcard**     | Перевод на сим-карту моб. оператора         |
| **alfa**        | Перевод строго между клиентами Альфа Банка  |
| **ozon**        | Перевод строго между клиентами Озон Банка   |
| **vtb**         | Перевод строго между клиентами ВТБ          |
| **sber**        | Перевод строго между клиентами СберБанка    |
| **psb**         | Перевод строго между клиентами ПСБ          |
| **uralsib**     | Перевод строго между клиентами Уралсиб      |
| **raiffeisen**  | Перевод строго между клиентами Райффайзен   |
| **yumoney**     | Перевод строго между клиентами ЮМани        |
| **mts**         | Перевод строго между клиентами МТС Банка    |
| **ak\_bars**    | Перевод строго между клиентами Ак Барс      |
| **sovkom**      | Перевод строго между клиентами Совкомбанк   |
| **wildberries** | Перевод строго между клиентами Вайлдберриз  |
| **solidarnost** | Перевод строго между клиентами Солидарность |
| **gasprom**     | Перевод строго между клиентами Газпром      |
| **yandex**      | Перевод строго между клиентами Яндекс       |
| **tbank**       | Перевод строго между клиентами ТБанк        |
| **rshb**        | Перевод строго между клиентами РСХБ         |
| **otp**         | Перевод строго между клиентами ОТП          |
| **nspk**        | Перевод по ссылке / QR-коду                 |
| **wire**        | Перевод по номеру счёта (реквизиты)         |
| **qr\_sber**    | Перевод по номеру счёта QR Сбер–Сбер        |
| **qr\_alfa**    | Перевод по номеру счёта QR Альфа–Альфа      |
| **qr\_ozon**    | Перевод по номеру счёта QR Озон–Озон        |
| **qr\_gasprom** | Перевод по номеру счёта QR Газпром–Газпром  |
| **qr\_otp**     | Перевод по номеру счёта QR ОТП–ОТП          |
| **qr\_vtb**     | Перевод по номеру счёта QR ВТБ–ВТБ          |
| **qr\_mts**     | Перевод по номеру счёта QR МТС–МТС          |
| **wire\_alfa**  | Перевод по номеру счёта Альфа–Альфа         |
| **wire\_kg**    | Перевод по номеру счёта Кыргызстан          |
| **card\_pdf**   | Перевод на карту с подгрузкой PDF чека      |
| **sbp\_pdf**    | Перевод на СБП с подгрузкой PDF чека        |

{% hint style="info" %}
Методы `card_pdf` и `sbp_pdf` требуют подгрузки чека после оплаты. Подробнее: [Загрузка чека](/orders/receipt-upload.md).
{% endhint %}

## Крипто-методы

| Метод           | Валюта | Сеть             | Описание                            |
| --------------- | ------ | ---------------- | ----------------------------------- |
| **usdt\_trc20** | USDT   | TRC20 (Tron)     | Перевод USDT в сети Tron            |
| **usdt\_erc20** | USDT   | ERC20 (Ethereum) | Перевод USDT в сети Ethereum        |
| **usdt\_ton**   | USDT   | TON              | Перевод USDT в сети TON             |
| **usdt\_bep20** | USDT   | BEP20 (BSC)      | Перевод USDT в сети BNB Smart Chain |
| **btc**         | BTC    | Bitcoin          | Перевод Bitcoin                     |
| **ltc**         | LTC    | Litecoin         | Перевод Litecoin                    |
| **eth**         | ETH    | ERC20 (Ethereum) | Перевод Ethereum                    |
| **bnb**         | BNB    | BEP20 (BSC)      | Перевод BNB в сети BNB Smart Chain  |
| **ton**         | TON    | TON              | Перевод TON                         |
