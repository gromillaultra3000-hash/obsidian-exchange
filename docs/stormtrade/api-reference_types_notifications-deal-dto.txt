Notifications\DealDTO

Объект Notifications\DealDTO представляет данные о сделке, включая информацию о статусе сделки, методах оплаты,
реквизитах, а также данные, связанные с возможными спорами и результатами платежа.

Описание полей объекта​

Название | Тип | Описание | 
id | string | Уникальный идентификатор сделки. | 
type | DirectionType | Направление платежа (входящий или исходящий). | 
status | DealStatus | Статус сделки. | 
paymentMethod | PaymentMethod | Метод оплаты, использованный для сделки. | 
paymentOption | PaymentOption | Вариант платежа, используемый для сделки. | 
requisites | RequisitesDTO | Реквизиты для проведения платежа по сделке. | 
canceledAt | string | Дата и время отмены сделки в формате ISO 8601. | 
disputeAttachment | string | null | Ссылка на прикрепленный файл с документами для разрешения спора, если применимо. | 
disputeReason | DisputeReason | null | Причина спора, если применимо. | 
disputeReasonData | DisputeReasonDataDTO | Дополнительные данные по причине спора. | 
payment | Notifications\PaymentDTO | null | Данные о платеже, если применимо. | 
attachments | string[] | Список ссылок на прикрепленные файлы. | 
disputeResult | DisputeResult | null | Результат спора, если применимо. |
