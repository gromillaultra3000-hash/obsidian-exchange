Коллбэки

Каждый раз, когда состояние инвойса изменяется, будет отправлен коллбэк в систему мерчанта с использованием POST запроса
на указанный notificationUrl в запросе Создание инвойса.

Тип уведомления определяется полем notificationType в JSON объекте запроса.
На данный момент поддерживается только один тип уведомления — invoice.

Каждый запрос будет содержать заголовок X-Notification-Token, в котором передается notificationToken (указанный при создании инвойса).
Если код состояния ответа не равен 200, система автоматически повторит попытку отправки
уведомления еще дважды с интервалом в 60 секунд между попытками.

Уведомление об изменения инвойса​

Это уведомление отправляется в случае изменения состояния инвойса. В теле запроса будет отправлен JSON объект
InvoiceNotificationDTO.
