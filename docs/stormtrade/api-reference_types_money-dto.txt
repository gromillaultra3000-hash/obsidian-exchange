MoneyDTO

Объект MoneyDTO используется для представления денежных сумм в различных валютах. Он включает информацию о величине
суммы, валюте и количестве знаков после запятой (субъединицах).

Описание полей объекта​

Название | Тип | Описание | 
amount | string | Сумма в основной валюте, представленная строкой для сохранения точности (например, “10.00000000”). | 
currency | string | Валюта счета в формате ISO 4217 (например, USDT). | 
subunit | integer | Количество десятичных знаков, используемых для представления дробных значений валюты. | 

Пример​

{
  "amount": "10.00000000",
  "currency": "USDT",
  "subunit": 8
}
