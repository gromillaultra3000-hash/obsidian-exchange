Интеграция "Host-To-Host"

Интеграция по модели “Host-To-Host” предполагает обмен данными между сервером мерчанта и сервером платежной системы.
Эта модель обеспечивает полную автоматизацию процессов приема платежей и осуществления выплат.

Прием платежей:
Сервер мерчанта в ответе API получает реквизиты, на которые клиент мерчанта должен совершить денежный перевод.
Ответственность за корректное отображение полученных реквизитов лежит на стороне мерчанта.

Осуществление выплат:
Для выполнения выплаты на клиентские реквизиты мерчант должен отправить API-запрос, передав реквизиты, на которые необходимо произвести оплату.
