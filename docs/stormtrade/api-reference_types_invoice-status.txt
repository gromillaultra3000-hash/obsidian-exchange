InvoiceStatus

Перечисление типа string которое содержит статус инвойса.

Возможные значения​

Значение | Описание | 
new | Инвойс только что создан. | 
paid | Инвойс был оплачен. | 
canceled | Инвойс был отменен. | 
dispute | Инвойс находится в стадии спора. | 
expired | Инвойс был отменен в связи с неоплатой |
