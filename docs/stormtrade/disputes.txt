Создание апелляций

Если клиент произвел оплату и подтвердил перевод, но по какой-то причине система не подтверждает сделку, клиент может открыть спор по данной сделке и прикрепить документ в качестве доказательства перевода средств.

Для создания апелляции необходимо отправить API запрос Открытие спора
