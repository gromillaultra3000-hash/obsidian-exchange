DirectionType

Перечисление типа string которое описывает направление платежей между конечными пользователями.

Возможные значения​

Значение | Описание | 
in | Прием - когда пользователь мерчанта осуществляет оплату по реквизитам сделки. | 
out | Выплата - когда пользователь мерчанта получает оплату на указанные реквизиты. |
