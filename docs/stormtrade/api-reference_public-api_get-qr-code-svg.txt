Получение SVG изображения QR кода

GET /api/public/invoices/{id}/qr-code

URL параметры​

Название | Тип | Описание | 
id | string | ID счета | 

Пример запроса​

CURL
curl --request GET \
  --url https://{{domain}}/api/public/invoices/3dbf4e92-ae60-4136-a7d6-bace925572dd1731483682/qr-code
